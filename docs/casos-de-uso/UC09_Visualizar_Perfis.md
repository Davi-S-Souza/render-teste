# UC14 – Visualizar Perfis

**Ator primário:** Usuário, Funcionário  
**Meta no contexto:** Permitir visualizar perfis de outros usuários.  

**Precondições:** O sistema deve possuir perfis cadastrados.  

**Disparador:** O ator acessa a opção “Visualizar Perfil”.  

**Cenário:**  
1. O sistema exibe dados públicos do perfil (nome, histórico de denúncias, avaliacao).  

**Exceções:**  

**Prioridade:** Baixa.  
**Quando disponível:** Incremento 3.  
**Frequência de uso:** Eventual.  
**Canal com o ator:** Interface web/mobile.
