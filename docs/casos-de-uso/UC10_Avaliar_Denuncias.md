# UC15 – Avaliar Denúncias

**Ator primário:** Funcionário  
**Meta no contexto:** Permitir que funcionários avaliem a pertinência de denúncias.  

**Precondições:** O sistema deve conter denúncias registradas.  

**Disparador:** O funcionário acessa a tela de avaliação.  

**Cenário:**  
1. O sistema exibe denúncia selecionada.  
2. O funcionário avalia relevância e prioridade.  
3. O funcionário aprova ou rejeita denúncia.  

**Exceções:**  

**Prioridade:** Importante.  
**Quando disponível:** Incremento 3.  
**Frequência de uso:** Frequente.  
**Canal com o ator:** Interface web.
