# UC10 – Acompanhar Status de Resolução

**Ator primário:** Usuário, Funcionário  
**Meta no contexto:** Permitir acompanhamento da resolução de uma denúncia.  

**Precondições:** A denúncia deve estar registrada.  

**Disparador:** O ator acessa os detalhes da denúncia.  

**Cenário:**  
1. O sistema exibe status atual da denuncia.  
2. Funcionário atualiza status quando necessário.  
3. O usuario clica no botao de sino na denuncia
4. Sistema notifica o denunciante e aqueles com o sino ativo sobre atualizaçoes.  

**Exceções:**  
- Nenhuma atualização → status permanece.  

**Prioridade:** Baixa.  
**Quando disponível:** Incremento 2.  
**Frequência de uso:** Frequente.  
**Canal com o ator:** Interface web/mobile.
