# UC06 – Filtrar Denúncias

**Ator primário:** Usuário, Funcionário  
**Meta no contexto:** Permitir aplicar filtros para refinar a visualização das denúncias.  

**Precondições:** O sistema deve possuir denúncias registradas.  

**Disparador:** O ator seleciona a opção de filtros.  

**Cenário:**  
1. O ator abre a tela de filtros.  
2. O sistema exibe opções de categoria, região e status.  
3. O ator seleciona filtros desejados.  
4. O sistema exibe os resultados filtrados.  

**Exceções:**  
- Nenhum filtro selecionado → sistema exibe lista completa.  

**Prioridade:** Importante.  
**Quando disponível:** Incremento 2.  
**Frequência de uso:** Frequente.  
**Canal com o ator:** Interface web/mobile.
