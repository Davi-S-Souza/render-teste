# UC11 – Gerenciar Denúncias

**Ator primário:** Funcionário  
**Meta no contexto:** Permitir que funcionários acompanhem e manipulem denúncias.  

**Precondições:** O usuário deve ser funcionário autenticado.  

**Disparador:** O funcionário acessa a tela de gerenciamento.  

**Cenário:**  
1. O sistema exibe denúncias registradas.  
2. O funcionário pode editar ou atualizar informações.  
3. O funcionario pode remover denuncia do ar

**Exceções:**  
- Tentativa inválida de atualização → sistema bloqueia ação.  

**Prioridade:** Essencial.  
**Quando disponível:** Incremento 2.  
**Frequência de uso:** Frequente.  
**Canal com o ator:** Interface web/mobile.
