# UC05 – Visualizar Denúncias

**Ator primário:** Usuário, Funcionário  
**Meta no contexto:** Permitir a visualização das denúncias registradas.  

**Precondições:** O sistema deve ter denúncias registradas.  

**Disparador:** O ator acessa a tela de denúncias.  

**Cenário:**  
1. O ator acessa a listagem de denúncias.  
2. O sistema exibe a lista de acordo com a ordenacao escolhida.  
3. O ator pode clicar em uma denúncia para ver detalhes.  

**Exceções:**  
- Nenhuma denúncia cadastrada → sistema exibe mensagem de vazio.  

**Prioridade:** Essencial.  
**Quando disponível:** Primeiro incremento.  
**Frequência de uso:** Diária.  
**Canal com o ator:** Interface web/mobile.
