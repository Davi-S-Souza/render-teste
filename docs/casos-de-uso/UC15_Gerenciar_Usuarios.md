# UC13 – Gerenciar Usuários e Conteúdo

**Ator primário:** Administrador do Sistema 
**Meta no contexto:** Permitir gerenciar usuários.  

**Precondições:** O ator deve estar autenticado como administrador.  

**Disparador:** O administrador acessa o painel de gerenciamento.  

**Cenário:**  
1. O sistema exibe lista de usuários,  
2. O administrador pode suspender usuários.  


**Prioridade:** Essencial.  
**Quando disponível:** Incremento 3.  
**Frequência de uso:** Ocasional.  
**Canal com o ator:** Interface web.
