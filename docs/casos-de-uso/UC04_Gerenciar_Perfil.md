# UC04 – Gerenciar Perfil

**Ator primário:** Usuário  
**Meta no contexto:** Permitir que o usuário visualize e edite suas informações de perfil.  

**Precondições:**  
- O usuário deve estar autenticado.  

**Disparador:** O usuário acessa a opção “Meu Perfil”.  

**Cenário:**  
1. O usuário acessa seu perfil.  
2. O sistema exibe os dados cadastrados.  
3. O usuário altera informações (nome, senha, foto).  
4. O sistema valida e salva os dados.  

**Exceções:**  
- Dados inválidos → o sistema solicita correção.  

**Prioridade:** Média.  
**Quando disponível:** Incremento 2.  
**Frequência de uso:** Eventual.  
**Canal com o ator:** Interface web/mobile.
