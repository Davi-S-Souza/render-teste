# UC07 – Consultar Mapa de Localização

**Ator primário:** Usuário, Funcionário  
**Meta no contexto:** Visualizar denúncias em mapa georreferenciado.  

**Precondições:** O sistema deve possuir denúncias registradas.  

**Disparador:** O ator seleciona a opção “Mapa”.  

**Cenário:**  
1. O sistema exibe o mapa da região.  
2. As denúncias aparecem como marcadores, considerando as opcoes de filtro escolhidas.  
3. O ator pode clicar em um marcador para ver detalhes.  

**Exceções:**  
- Nenhuma denúncia cadastrada → mapa exibido vazio.  

**Prioridade:** Baixa.  
**Quando disponível:** Futuramente.  
**Frequência de uso:** Frequente.  
**Canal com o ator:** Interface web/mobile.
